# ChatGPT Issue navigation

为 `https://chatgpt.com/*` 对话页添加一个“问题目录”悬浮侧栏，把当前会话中的用户提问提取出来，便于快速跳转。

## 功能

- 自动扫描当前会话中的用户消息
- 按提问顺序生成目录列表
- 支持关键词搜索问题
- 点击目录项后平滑滚动到对应提问
- 跟随页面滚动高亮当前所在问题
- 支持拖动面板位置，并记住折叠状态与位置

## 文件

- `index.js`: Tampermonkey 用户脚本主体

## 安装

1. 安装 Tampermonkey。
2. 新建脚本并粘贴 `index.js` 内容。
3. 保存后打开任意 `chatgpt.com` 对话页。

## 使用

- 页面加载完成后，右侧会出现“问题目录”面板。
- 输入框可按问题文本过滤目录项。
- 点击“折叠/展开”可收起或展开面板。
- 拖动标题栏可调整面板位置。

## 实现说明

- 通过 `article[data-testid^="conversation-turn-"]` 查找对话轮次。
- 仅提取 `data-turn="user"` 的消息作为目录项。
- 使用 `MutationObserver` 监听对话变化，在新消息出现后自动刷新目录。

## 注意事项

- 该脚本依赖 ChatGPT 当前页面 DOM 结构；如果站点改版，选择器可能需要更新。
- 面板默认会在页面加载后约 1.2 秒初始化。
