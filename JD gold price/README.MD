# JD gold price

为 `https://dingpan.jd.com/*` 页面添加一个顶部控制面板，用于记录日志、捕获交易请求参数，并在已获取参数后快速执行下单或出货请求。

## 功能

- 页面顶部显示日志区与操作按钮
- 捕获页面中的实时价格与 `priceUuid`
- 捕获买入与卖出请求的 URL、方法、请求体和请求头
- 基于最新价格重放下单/出货请求
- 可开启自动模式，在失败后按固定间隔自动重试
- 自动拦截部分交易失败提示弹窗
- 成功后自动查询当前持仓总克重

## 文件

- `index.js`: Tampermonkey 用户脚本主体

## 面板按钮

- `清理日志`: 清空顶部日志
- `自动模式: 开/关`: 切换失败后自动重试
- `出货`: 发送卖出请求
- `下单`: 发送买入请求

## 使用前提

- 脚本要先从页面真实请求中捕获必要参数。
- 也就是说，通常需要先在页面里完成过一次对应流程，脚本才能拿到买入或卖出的请求凭证。
- 如果还没有捕获到完整参数，脚本会在日志里提示错误。

## 安装

1. 安装 Tampermonkey。
2. 新建脚本并粘贴 `index.js` 内容。
3. 保存后打开 `dingpan.jd.com` 对应页面。

## 实现说明

- 通过重写 `XMLHttpRequest.prototype.open/send/setRequestHeader` 记录关键请求信息。
- 从 `stdLatestPrice` 响应中提取最新价格和价格 ID。
- 从 `stdRechargeAndBuyGold` 与 `stdSellGold` 请求中记录交易所需参数。
- 重放请求前会把最新价格写回请求参数中，再发起实际交易请求。

## 注意事项

- 该脚本运行在真实交易页面，使用前应自行确认平台规则、请求语义和潜在风险。
- 页面接口、参数格式或弹窗结构变化后，脚本可能失效。
- 自动模式会在失败后继续重试，开启前应确认自己的预期行为。
