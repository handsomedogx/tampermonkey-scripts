# Shanbay space to next word

为 `https://web.shanbay.com/*` 学习页面增加空格快捷键，按下空格即可触发“下一词”按钮。

## 功能

- 监听空格键并点击下一词按钮
- 兼容页面按钮被重新渲染的情况
- 在输入框、文本域、可编辑区域聚焦时不拦截空格
- 自动给匹配到的按钮追加 `Space` 提示

## 文件

- `index.js`: Tampermonkey 用户脚本主体

## 安装

1. 安装 Tampermonkey。
2. 新建脚本并粘贴 `index.js` 内容。
3. 保存后打开扇贝网页学习页面。

## 使用

- 在学习页面中，非输入状态下按空格会触发“下一词”。
- 如果当前焦点在输入框、文本域或可编辑区域，脚本不会接管空格键。

## 实现说明

- 使用前缀选择器匹配 `StudyPage_nextBtn__` 开头的按钮类名，适配构建产物哈希变化。
- 通过 `MutationObserver` 监听 DOM 变化，保证按钮重建后仍能继续使用。
- 触发点击时会依次发送 `mousedown`、`mouseup` 和 `click`，尽量模拟真实交互。

## 注意事项

- 如果扇贝调整页面结构或按钮类名规则，选择器可能需要同步修改。
